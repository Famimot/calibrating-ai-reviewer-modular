КРИТЕРИЙ: ПРАКТИЧЕСКАЯ ЦЕННОСТЬ (20 баллов)
ГЛАВНОЕ ПРАВИЛО:
Оценивается не только то, на чём автор продемонстрировал метод, но и потенциал применения в других задачах / дисциплинах, если автор это явно заявил или это очевидно следует из логики метода.
Запрещено:
* Снижать оценку за то, что автор выбрал узкий пример для демонстрации широкого метода
* Требовать готового промышленного внедрения

Определи, в скольких дисциплинах / типах задач / контекстах можно применить результат.
Баллы
Уровень
Описание
Маркеры в тексте
8–10
Междисциплинарный
Применим в 3+ разных дисциплинах или классах задач
«метод может быть применён в…», «обобщается на…», «потенциально используется в биологии, медицине, экономике…»
6–7
Умеренно широкий
Применим в смежных областях (2–3 дисциплины)
«метод может быть использован в [смежная область 1] и [смежная область 2]»
3–5
Узкий
Применим только в своей узкой нише / типе задач
Примеры только из одной предметной области, без указания на обобщение
0–2
Единичный случай
Применим только к конкретному объекту / набору данных
Нет обобщений, всё завязано на уникальные характеристики примера
Правило приоритета:
* Если автор явно указал потенциал обобщения (фразы выше) ? оценивай по заявленному потенциалу
* Если автор не указал, но из метода это очевидно (например, математическая формула без привязки к предмету) ? можно повысить на 1–2 балла, но с пометкой «не заявлено, но очевидно»
* Если метод жёстко привязан к предмету ? оценивай по факту

ШАГ 2. ОЦЕНКА ГЛУБИНЫ / ЗНАЧИМОСТИ (0–10 баллов)
Определи, насколько велик потенциальный эффект от применения.
Баллы
Уровень
Описание
Маркеры в тексте
9–10
Парадигмальный сдвиг
Открывает новый класс задач, меняет практику в области, создаёт новое направление
«впервые позволяет…», «открывает возможность для…», «меняет подход к…»
6–8
Существенное улучшение
Значительный прирост точности / скорости / надёжности / информативности
улучшение на 20%+, сравнение с аналогами в пользу нового метода, решение ранее нерешаемой задачи
3–5
Инкрементальное
Небольшое улучшение или новое применение старого метода
улучшение до 20%, удобная демонстрация, частный случай известного подхода
0–2
Тривиальное / отсутствует
Ничего не меняет, результат очевиден заранее, практической пользы не описано
«требуются дальнейшие исследования», «потенциально может быть полезно» без конкретики
Правила:
* Для I-Novelty: значимость обычно ниже (3–8 баллов), так как это развитие существующего
* Для P-Novelty: значимость может быть высокой (6–10), если метод действительно новый
* Если автор не привёл количественных сравнений — не вычитай автоматически, оценивай по качественному описанию эффекта
