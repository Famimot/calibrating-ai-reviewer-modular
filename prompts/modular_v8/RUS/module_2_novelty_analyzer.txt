## ОПРЕДЕЛЕНИЕ ТИПА СТАТЬИ (выполняется первым)
## Вопрос 1 чек-листа P-Novelty (ДА/НЕТ)

**Примечание по типу публикации:**
Правило применяется ко всем типам публикаций, включая монографии.
Если в монографии описан собственный метод/теория — применяется полная шкала P-Novelty (23–30) без занижения.

**Формулировка:**
Статья заявляет принципиально новый ИЛИ развивает собственный ранее опубликованный метод/теорию, а не адаптирует известное чужое решение под другой объект или условия.

**ДА, если выполняется хотя бы одно:**
- Решается значимая задача для науки или практики задача (с обоснованием значимости в статье), а не локальная для конкретного случая
- Предложена принципиально новая физика/математика/алгоритм
- Решается новый класс задач, ранее не решавшихся
- Предложен принципиально иной подход, меняющий понимание проблемы
- Развитие/усовершенствование собственного ранее опубликованного метода (нетривиальное, дающее новое качество)

**НЕТ, если:**
- Метод основан на небольших/непринципиальных изменениях чужого метода
- Выполнен перенос известного метода на другой объект без концептуальных изменений
- Применение ранее известного метода в других условиях 
- Тривиальная адаптация своего же метода (например, смена входных параметров без изменения логики)

## Вопрос 2 чек-листа P-Novelty (ДА/НЕТ)

**Формулировка:**
Материалы для воспроизведения (код/данные/промпт/протокол) открыто доступны?

**ДА, если:**
- Прямая ссылка на репозиторий (GitHub, Zenodo, Figshare, Harvard Dataverse)
- DOI исходных материалов
- Прямая ссылка на Supplementary materials в журнале

**НЕТ, если:**
- Упоминание «доступно по запросу у автора»
- Материалы не описаны
- Описаны, но не доступны

### Таблица баллов P-Novelty (0–30)

| Вопрос 1 | Вопрос 2 | Уровень | Балл |
|----------|----------|---------|------|
| ДА | ДА | Высший | 27–30 |
| ДА | НЕТ | Пониженный | 23–24 |

**Дополнительное правило:**
- Если метод является развитием собственного ранее опубликованного (нетривиальное), а не первой публикацией → **-1 балл** от максимума

**Распределение внутри диапазона:**
- 27–30: материалы полностью открыты и достаточны + метод значим для науки
- 23–24: метод новый, но материалы закрыты или неполны


### I-NOVELTY 

**Примечание по типу публикации:**
Правило применяется ко всем типам публикаций, включая монографии.
Если в монографии используются чужие методы — применяется полная шкала I-Novelty (0–22) без занижения.

**Применяется, когда:**
- Вопрос 1 P-Novelty = НЕТ (в статье используются ранее известные чужие (не авторские) методы для получения новых данных)
- Тип статьи = оригинальное исследование

**Критерии оценки (0–22 балла):**

| Баллы | Уровень | Описание |
|-------|---------|----------|
| 18–22 | Высокий | Развитие/адаптация чужих концепций с получением новых нетривиальных данных; глубокая интерпретация |
| 13–17 | Средний | Новые данные на базе известного метода, но без концептуального развития |
| 7–12 | Низкий (тривиально) | Тривиальное применение метода к новому объекту; результаты ожидаемы |
| 0–6 | Очень низкий / нет | Отсутствие новых данных; описательная работа без результатов |

**Учёт режима (STEM/Humanities):**
- STEM: новизна в экспериментальных данных, алгоритмах, моделях
- Humanities: новизна в интерпретации, контексте, архивных источниках

### R-NOVELTY (обзор / краткое сообщение)

**Применяется, когда:**
- Тип статьи = обзор, краткое сообщение, technical note (не монография)
- Не применяется для монографий (для них действуют правила P-Novelty или I-Novelty в зависимости от наличия авторского метода)

**Критерии оценки (0–15 баллов):**

| Баллы | Уровень | Описание |
|-------|---------|----------|
| 13–15 | Высокий | Новая классификация / выявление значимого пробела / новая программа исследования / систематический обзор с мета-анализом |
| 8–12 | Средний | Систематический обзор с выводами / выявлены направления для исследований / краткое сообщение с новой идеей |
| 4–7 | Низкий | Нарративный обзор без структуры / краткое сообщение без методологии / перечисление работ с минимальным анализом |
| 0–3 | Очень низкий | Формальный обзор / тривиальная заметка / отсутствие вклада |

**Учёт режима (STEM/Humanities):**
- STEM: новизна в систематизации данных, мета-анализе, выявлении пробелов в экспериментальных методах
- Humanities: новизна в периодизации, типологии источников, выявлении пробелов в теории
