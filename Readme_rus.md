# Калибровочный ИИ-рецензент: модульная архитектура

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.xxxxxx.svg)](https://doi.org/10.5281/zenodo.xxxxxx)

## О проекте

Данный репозиторий содержит промпты, программное обеспечение и результаты валидации модульной архитектуры калибровочного ИИ-рецензента. Архитектура состоит из шести независимых модулей, последовательно реализующих определение режима рецензирования (STEM/Humanities), оценку научной новизны, методологическую строгость, практическую ценность, качество визуализации и воспроизводимость.

Подробное описание метода и результаты валидации представлены в статье:

> Кравцов Г.Г. Калибровочный ИИ-рецензент: модульная архитектура для оценки и ранжирования научных статей // Научно-техническая информация. 2025.

## Использование

### Использование промптов

1. Откройте новый диалог с LLM (рекомендуется DeepSeek-V3)
2. Загрузите Модуль 1 с выбранным режимом (USER_MODE = 0 или 1) и полные тексты статей (рекомендуется не более 7). Выполните первый запрос модуля.
3. Далее последовательно, после получения результатов предыдущего модуля, загружайте Модули 2–6

### Использование программного обеспечения

1. Скачайте `llm_review_calibration_eng.exe` или `llm_review_calibration_eng.py`
2. Запустите программу и загрузите файл `results/Appendix_C_Modular_10_runs_raw_data.xlsx`
3. Программа автоматически выполнит анализ 10 прогонов, построит таблицы и графики, сохранит результаты в Excel

**Требования для Python-скрипта:**
- Python 3.8+
- Установленные зависимости: `pandas`, `numpy`, `matplotlib`, `openpyxl`

## Воспроизводимость

Любой пользователь может:
1. Взять произвольные рукописи (не менее 2)
2. Использовать промпты из этого репозитория
3. Повторить алгоритм сопоставления и ранжирования статей
